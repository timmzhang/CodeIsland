评审结论 · 2026-07-11 全部拍板

5 个决策点 — 已定稿

cache 读要不要计入主图？ ✅ 已拍板

主图只画 input + output，cache 读写进汇总卡和明细表（cache 读单价约为 input 的 1/10、量级又大 10 倍以上，混入主图柱子会失真）。本稿即此口径。

成本数字怎么标？ ✅ 已拍板

统一叫「等效 API 成本」并加脚注（订阅用户实际不按此付费，它表示"同样用量走 API 要花多少"）——同时是"订阅回本了多少"的爽点。

L1 徽标常驻还是悬停出现？ ✅ 已拍板

默认常驻（数字 + 迷你柱约 90pt 宽），设置里可关。

统计窗口入口放哪？ ✅ 已拍板

放在展开面板右上角：下划线文字「今日 token 燃烧 2.03M」，点击打开统计窗口（参考竞品截图样式，2026-07-11 拍板）。L1 mock 已按此更新。

统计不到的工具（Cursor / Copilot 等 IDE 类）怎么表达？ ✅ 已拍板

不在图表中出现，仅在明细表脚注声明「未列入」。
